to put images on a page you use the img tag

we can make images responsive to things like the size of the screen. This is helpful because even though you can resize images with css making images responsive means that you can choose which image the browser downloads for smaller screens that download smaller images this can improve performance size

here I have a set of images that I am going to choose from depending on the screen size

using a srcset allows us to give the images to pick from to the browser along with their size (width in this case) and the browser will pick depending on configurable factors (in this case the view port width)

we can also use teh picture element to get more functionality here the picture element is using a media query to choose between the images. Using picture elements is good practice but you still have to support older browsers and include the default img
